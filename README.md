## Getting Started

### Requirements

* iOS 15.0+
* Xcode 13.0+
* Swift 5.5+

### Project Setup

Run the shell script and download the third party modules.

```sh
chmod +x setup.sh
./setup.sh
```

## Usage

1. Enable Extension in Safari.
2. Go to any URL and select `Develop Menu` from the Safari menu.
